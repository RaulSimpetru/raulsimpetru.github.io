3D Brain Visualiser

Language: Python

For this project, I tried to recreate and display a brain in 3D space using OpenGL from a lot of 2D brain slices from this kaggle dataset.

From the slices, I get the biggest contour possible using OpenCV and turn every 2D contour in a 3D slice.

By stacking them over each other I was able to display the reconstructed brain.
